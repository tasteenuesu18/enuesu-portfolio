# enuesu Portfolio

ARホログラムのようなUIと没入感のある演出を取り入れたポートフォリオサイト。
空間に浮かび上がる情報と、洗練されたダークテーマの世界観を表現しています。

## 🚀 技術スタック (Main Stack)

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **3D / WebGL**: Three.js / React Three Fiber
- **Animation**: Framer Motion / GSAP
- **Deployment**: Cloudflare Pages

## 🎨 Design Concept

**"AR Holo Interface in the Void"**

深淵（Void）に浮かぶホログラフィックなインターフェース。
SF映画や近未来的なデバイスからインスピレーションを受けた、実用的かつ没入感のあるデザインを目指しました。
